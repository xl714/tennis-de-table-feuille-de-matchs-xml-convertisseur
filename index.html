Résumer une feuille de match XML d'une rencontre par équipe

Dans le logicial GIRPE, à la fin d'une rencontre par équipe. Envoyez vous la feuille de match au format XML

Ensuite téléchargez le fichier sur votre ordinateur.

    Dans le formulaire ci-dessous sélectionnez le fichier.

  • Entrez ensuite le numéro de votre équipe
